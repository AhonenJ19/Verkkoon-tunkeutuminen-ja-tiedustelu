## x.) Tiivistelmät
- Pyramidi näyttää eri indikaattorityypit sekä kuinka paljon hyökkääjälle aiheutuu ongelmia, kun ne estetään
- Kertoo siis erilaisten uhkien arvot, mitä kannattaa estää
- Timanttimallin päätavoitteena on tunnistaa hyökkääjät, ymmärtää niiden tavoite, taktiikat, motivaatio
- Auttaa organisaatioita analysoimaan hyökkäyksiä


## a.) Apache log

Asennetaan Apache ja käynnistetään se
```sudo apt install apache2 -y```  ```sudo systemctl start apache2```

Avataan selaimessa ```http://localhost```


<img width="1194" height="819" alt="Screenshot 2026-04-10 125058" src="https://github.com/user-attachments/assets/00be4b0c-278f-43d5-aae2-459e0f0e3d6e" />


Etsitään lokirivi ```sudo cat /var/log/apache2/access.log```

<img width="951" height="116" alt="image" src="https://github.com/user-attachments/assets/7ab58e6e-9091-4f63-aa6b-cfeaadae69cf" />

Rivillä näkyy oma ip osoite ```127.0.0.1``` sekä aikaleima

Get pyyntö sekä protokolla ```"GET / HTTP/1.1"``` 

Statuskoodi 200, eli onnistunut sekä vastauksen pituus tavuina ```200 3383```

Viimeinen kohta on User agent, selaimen tunniste ja käyttöjärjestelmä ```"Mozilla/5.0 (X11; Linux x86_64; rv:140.0) Gecko/20100101 Firefox/140.0"```


## b.) Nmapped

<img width="1438" height="427" alt="image" src="https://github.com/user-attachments/assets/d525b879-5731-49bd-ad54-516fb74256e1" />

Ohjelmiston versio ```Apache httpd 2.4.66 ((Debian))```

Oletussivu ```http-title: Apache2 Debian Default Page: It works```

Käytetty laite, tavallinen kone ```Device type: general purpose```

Käyttöjärjestelmä, versio ```Running: Linux 2.6.X / 5.X``` ```OS details: Linux 2.6.32, Linux 5.0 - 6.2```

Kertoo hyppyjen määrän, 0 koska kohde on sama kone ```Network Distance: 0 hops```

## g.) Agentti
